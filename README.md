## How to publish with GitHub Pages

1. Open repository Settings.
2. Go to Pages.
3. Under Build and deployment, choose `Deploy from a branch`.
4. Branch: `main`.
5. Folder: `/root`.
6. Save.
7. Wait 1–5 minutes.
8. Open: `https://hanifbhuian.github.io/PSD-Survey/`

## Note

The actual survey form and Google Sheet submission are handled by Google Apps Script. This GitHub Pages site is a landing page to make public sharing smoother and improve social preview.
